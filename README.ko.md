## 개요

Konado는 다양한 대화 템플릿과 강력한 대화 관리자를 제공하는 경량 비주얼 노벨 엔진 프레임워크입니다. 유연성에 중점을 두고 설계되어 있어 개발자는 대화 중심 게임을 빠르게 만들거나 기존 게임 프로젝트에 대화 시스템을 원활하게 통합할 수 있습니다.
