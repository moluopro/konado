## Aperçu

Konado est un kit d'outils de création de dialogues pour Godot Engine, avec des modèles et un gestionnaire de dialogues pour vous aider à construire rapidement des romans visuels, des jeux gal, des RPG et d'autres projets axés sur l'histoire.

## Démarrage Rapide

Mettez Konado en place en quelques minutes en suivant notre [Guide de Démarrage Rapide](https://godothub.com/oss/konado/tutorial/install.html).

## Documentation

Pour des guides complets, des références API et des tutoriels avancés, visitez notre site web officiel du projet : https://godothub.com/oss/konado.
