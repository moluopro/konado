## 簡介

Konado 是一套視覺小說引擎框架，提供多種對話模板與對話管理器，可快速建立對話遊戲，也能嵌入各類遊戲的對話場景中。


## 快速入門

請參閱 [快速入門](https://godothub.com/oss/konado/tutorial/install.html) 文件，瞭解如何快速開始使用 Konado。
