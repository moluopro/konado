# 播放音效

## 功能概述
用于在特定场景或交互节点中播放指定的音效文件，增强场景氛围和交互反馈，播放音效是即时执行的，不会阻塞后续脚本执行。

## 语法规范
```
play sfx <音效名称>
```

## 参数说明
| 参数       | 是否必需 | 示例         | 说明                     |
|------------|----------|--------------|--------------------------|
| 音效名称   | 是       | door_open    | 需播放的音效文件标识名   |

## 代码示例
```
play sfx door_open
play sfx button_click
play sfx notification
```
