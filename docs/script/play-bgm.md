# 播放BGM

## 功能概述
用于在特定场景或交互节点中播放指定的背景音乐（BGM），用于营造场景氛围和情绪渲染，播放BGM是即时执行的，不会阻塞后续脚本执行。

## 语法规范
```
play bgm <音乐名称>
```

## 参数说明
| 参数       | 是否必需 | 示例         | 说明                     |
|------------|----------|--------------|--------------------------|
| 音乐名称   | 是       | main_theme   | 需播放的BGM文件标识名    |

## 代码示例
```
play bgm main_theme
play bgm peaceful_morning
play bgm tension_theme
```
