# 建議與回饋

如果您對 `Konado` 專案的文件、開發流程、具體程式碼等內容有任何建議，或是發現了文件或程式碼中的錯誤，都歡迎您向我們回饋。

如果願意以任何方式參與進來，或者為我們提供協助，請[聯絡我們](#建議管道)。作為一個社群協作的開源專案，您的貢獻會讓 `Konado` 變得更好。


## 問題回報

向官方儲存庫[提交 issue](https://github.com/godothub/konado/issues)，分為以下三種類型：

- **Bug 回報**：描述問題，需包含重現步驟
- **功能建議**：在 Issue 區描述需求背景和預期效果
- **文件改進**：在 Issue 區描述需要改進的地方
